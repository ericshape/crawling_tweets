# crawling tweets
> crawling tweets via specific user IDs

## Getting started

1. add Twitter API keys and secrets to ```twitterOauth.py```
2. add user IDs to ```user_list.csv```
3. run ```getTweets.py``` in the shell. 
```shell
python getTweets.py user_list.csv
```
